# ObjectC_cast
轉型（隱性或稱自動轉型, 顯性或稱強制轉型）

程式設計師常常遇到設定運算子右邊的資料型別和左邊的資料型別不一致的情況，所以要進行料型別轉換。
『資料轉型』能將設定運算子左右兩邊資料型別轉為相同。
